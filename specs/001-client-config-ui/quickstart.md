# Quickstart: Client-Side Config UI

**Feature**: [specs/001-client-config-ui/spec.md](specs/001-client-config-ui/spec.md)
**Plan**: [specs/001-client-config-ui/plan.md](specs/001-client-config-ui/plan.md)
**Date**: 2026-02-18

## Steps

1. Checkout branch `001-client-config-ui`
2. Install dependencies: `npm install`
3. Add config page: `pages/config.vue`
4. Implement UI for calendar/quote URLs
5. Store config in localstorage
6. Remove old install/startup scripts
7. Add new `startup-kiosk.sh` script (see research.md)
8. Test config UI (Vitest, Playwright)
9. Test kiosk script on Raspberry Pi
